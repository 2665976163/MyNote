MVC  与 三层架构 的区别

MVC：
         MVC是由Model[模型] View[视图] Controller[控制器]组成

	Model 模型层：
		      主要处理控制器转发过来的请求
	Controller 控制器：
		       主要将用户的请求转发交给给与之对应的模型
	View 视图层：
	 	      主要编写用户请求(html 、css 、js 、jsp 、jquery等)

三层架构：
	三层架构是由 View [视图]  Server [服务]  Dao [持久] 组成
	
	View 视图层：
		 主要编写用户请求(html 、css 、js 、jsp 、jquery等)与对应的后台处理
	Server 服务层：
		主要处理用户请求的后台处理，将Dao 持久层 中的数据组成与之对应的处理方法
	Dao 持久层：
		直接操作数据库，一些原子操作.
