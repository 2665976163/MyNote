> 概念

```java
异常链是一种面向对象编程技术，指将捕获的异常包装进一个新的异常中并重新抛出的异常处理方式。
```

> 正规

```java
public class Main {
	public static void main(String[] args){
		Main main = new Main();
		try {
			main.test1();
		}catch(RuntimeException e) {
			e.printStackTrace();
		}
		System.out.println("继续执行");
	}
	public void test1() {
		try {
			test2();
		}catch(ArithmeticException e) {
			RuntimeException ex = new RuntimeException("不想**，但是没有办法");
			ex.initCause(e);
			throw ex;
		}
	}
	public void test2() {
		try {
			int a = 1/0;
		}catch(ArithmeticException e) {
			throw new ArithmeticException("**..");
		}
	}
}
```

> 利用上篇笔记上自定义的异常创建一个异常链

```java
import java.util.Scanner;

public class Main{
    Scanner input = new Scanner(System.in);
    static Main model = new Main();
    public static void main(String args[]){
        try{
            model.test1();
        }catch(RuntimeException ex){
            ex.printStackTrace();
        }
    }
    public void test1(){
        try{
            test2();
        }catch(MyException ex){
            RuntimeException exs = new RuntimeException("底层代码出错！",ex);
            throw exs;
        }
    }
    public void test2(){
        System.out.println("值：");
        int select = input.nextInt();	//输入的是0
        try{
            test3(select);
        }catch(ArithmeticException ex){
            MyException exs = new MyException("参数传错了啊，靓仔",ex);
            throw exs;
        }
    }
    public void test3(int value){
        try{
            int temp = 1/value;
        }catch(ArithmeticException ex){
            ex = new ArithmeticException("计算错误soory，不能除以0哦");
            throw ex;
        }
    }
}
```
