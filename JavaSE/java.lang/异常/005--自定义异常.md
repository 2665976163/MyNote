## 自定义异常

> 概念

```java
如果想写一个检查性异常那么就继承 "Exception" 运行时异常则继承 "RuntimeException".
```

> 举例  编译时异常 | 检查性异常

```java
public class MyException extends Exception{
    MyException(){}
    MyException(String message){
        super(message);	//附带用户消息
    }
    MyException(Throwable cause){
        super(cause);	//引发这个异常的异常.异常链生成的第二种方式
    }
    MyException(String message,Throwable cause){
        super(message,cause);	//附带用户信息 + 引发这个异常的异常
    }
}
```

```java
以上是个检查时异常，如果有方法抛出或声明该异常，那么调用者必须处理  捕获 | 声明
```

> 举例  运行时异常

```java
class MyException extends RuntimeException{
    MyException(){}
    MyException(String message){
        super(message);	//附带用户消息
    }
    MyException(Throwable cause){
        super(cause);	//引发这个异常的异常.异常链生成的第二种方式
    }
    MyException(String message,Throwable cause){
        super(message,cause);	//附带用户信息 + 引发这个异常的异常
    }
}
```
